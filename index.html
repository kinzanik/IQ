Понятие квадратного уравнения Уравнение — это математическое равенство, в котором неизвестна одна или несколько величин. Значение неизвестных нужно найти так, чтобы при их подстановке в пример получилось верное числовое равенство. Например, возьмем выражение 8 + 4 = 12. При вычислении левой части получается верное числовое равенство, то есть 12 = 12. Уравнением можно назвать выражение 8 + x = 12, с неизвестной переменной x, значение которой нужно найти. Результат должен быть таким, чтобы знак равенства был оправдан, и левая часть равнялась правой. Степень уравнения можно определить по наибольшей степени, в которой стоит неизвестное. Если неизвестное стоит во второй степени, значит, такое уравнение является квадратным. Квадратное уравнение — это ax2 + bx + c = 0, где a — первый или старший коэффициент, не равный нулю, b — второй коэффициент, c — свободн
